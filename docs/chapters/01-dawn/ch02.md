# 第2章　几位奠基者与早期探索

> "苏步青先生说，数学要为工业服务。这句话，我们那一代人是当真的。"

---

## 2.1 苏步青：从工业需求到学科远见

苏步青（1902–2003）是中国最重要的数学家之一，也是计算几何在中国萌芽时期绕不开的名字。他在复旦大学长期主持数学工作，以微分几何见长，但他从未将数学视为象牙塔中的事业。

1970年代，苏步青率先提出将数学方法应用于船体放样，推动复旦数学系与造船工业建立合作。他的远见不仅在于解决具体问题，更在于他意识到：**计算机辅助设计将成为工业数学的主战场**，中国必须培养这方面的人才。


## 2.2 复旦的早期实践

在苏步青的推动下，复旦大学数学系在1970年代末开始开展计算几何相关研究。刘鼎元等人参与了早期的样条函数研究和数控编程工作，为日后建立完整的计算几何课程体系奠定了基础。

复旦的路径偏向**理论引入与教学传播**：率先开设课程，编写讲义，培养学生，把西方的前沿成果系统地引入国内。

## 2.3 山东大学：扎根工业的传统

山东大学与青岛造船业的地缘关系，使其自然成为船体放样研究的重要基地。汪嘉业等人在这里建立起了一支工程导向鲜明的研究队伍。

山大的风格与复旦不同，更强调**从工程问题出发**，以参数曲线与曲面的计算实现为核心，注重系统性的软件实现。

## 2.4 浙江大学：数学传统的加持

浙江大学的数学系有着深厚的分析和逼近论传统。当梁友栋等人开始关注计算几何时，这种数学底蕴使他们能够以更严格的理论框架审视这一领域。

浙大的早期探索集中在**样条函数的理论性质**以及与逼近论的联系上，这一路径日后演变为浙大学派鲜明的数学风格。

## 2.5 样条、逼近与数控绘图的最初尝试

这一时期中国学者的研究，主要集中在以下几个方向：

| 方向 | 代表机构 | 核心问题 |
|------|---------|---------|
| 三次样条插值 | 复旦、浙大 | 给定型值点，构造光滑插值曲线 |
| 船体曲面拟合 | 山东大学 | 离散测量数据的曲面重建 |
| 数控编程辅助 | 多所高校 | 曲线数据转化为机床指令 |
| B 样条初步 | 浙江大学 | 引入 B 样条的理论性质 |

这些探索彼此分散，缺乏交流，研究者们甚至不知道国内还有人在做类似的工作。这一状况，即将因为一个人的归来而改变。
