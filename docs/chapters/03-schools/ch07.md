# 第7章　山东大学：从船体放样到系统实现

---

## 7.1 汪嘉业与山大的工业传统

山东大学计算几何的发展，与青岛的造船工业有着天然的地缘联系。汪嘉业长期在山大主持计算几何研究，其团队的工作始终与工程实践保持紧密联系。

山大的风格是务实的：不过多纠缠于理论的精细证明，而是强调**算法的实现性和工程的可用性**。

## 7.2 参数曲线、工程导向与人才培养

山大的研究重点集中在：
- 参数曲线曲面的高效计算算法
- 船体曲面的数值拟合与光顺处理
- 从数学结果到实际软件系统的转化

汪嘉业团队培养了大量以工程应用为导向的研究生，许多人日后进入工业界，成为 CAD/CAM 领域的骨干力量。

## 7.3 "从生产中来，到生产中去"

山大计算几何的文化基因，可以用这句话来概括。这不只是一种方法论，也是一种价值观：数学的价值要在解决真实问题中得到体现。

这种务实精神，使山大与复旦、浙大形成了互补的格局，共同支撑起协作组的多元生态。
