# 第13章　2001：GDC 专委会的成立

> "从'协作组'到'专委会'，不只是名字的改变，是一代人完成了交接。"

---

## 13.1 为什么要从"协作组"走向"专委会"

进入2000年代，全国计算几何协作组运行了将近二十年，已经显现出一些结构性的问题：没有正式的组织架构，没有稳定的资金来源，活动的连续性高度依赖少数核心人物的个人投入。

与此同时，中国计算机学会（CCF）的学科分支体系日趋完善，为建立更正式的专业委员会提供了制度框架。2001年前后，在多方推动下，**几何设计与计算专委会（GDC）**正式成立，挂靠中国工业与应用数学学会（CSIAM）。

## 13.2 清华会议与学科共识的形成

GDC 专委会成立前后，在清华大学举办了一次重要的学术会议。这次会议不只是一次学术交流，更是新老两代学者之间的一次**学科共识确认**：

- 计算几何在中国有独立存在的价值，不应被完全并入图形学
- 几何设计的工业应用方向（CAD/CAM）仍然是核心
- 需要吸引新一代学者进入，形成梯队

这次会议，某种程度上是1982年青岛短训班的精神续集——一次新的出发。

## 13.3 新的组织架构与运作方式

GDC 专委会相比协作组，在组织上更为规范：
- 有明确的委员会成员名单和任期制度
- 举办正式的全国学术年会（GDC 会议）
- 与国际学术组织建立联系
- 推动出版专业教材和学术论著
