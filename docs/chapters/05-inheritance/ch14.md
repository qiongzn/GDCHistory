# 第14章　新一代学者与研究格局

---

## 14.1 清华大学：胡事民、唐泽圣等

进入2000年代，清华大学计算机系成为计算几何与图形学的重要新兴力量。胡事民在网格处理、几何建模方面的研究在国际上引起关注；唐泽圣在计算机图形学的系统研究方面有深厚积累。

清华的加入，为 GDC 专委会带来了新的学术风格：**更靠近计算机科学，更重视国际发表，更关注新兴的几何计算问题**。

## 14.2 浙江大学的持续影响

浙大在2000年代继续保持了在计算几何领域的核心地位。汪国昭、王国瑾等人的研究持续推进，同时培养了大批新一代学者。浙大的学术谱系，在这个阶段进一步向全国扩散。

## 14.3 中科院计算所的新角色

中国科学院计算技术研究所在2000年代承担了更多以**工程系统**为导向的几何计算研究，成为连接学术研究与产业应用的重要节点。

## 14.4 吉林大学、中科大等单位的加入

研究格局在这一阶段明显扩展，越来越多的高校加入到 GDC 的学术社区：
- **吉林大学**：在 CAD 几何核心和工业软件方向有持续投入
- **中国科学技术大学**：在几何处理和图形学基础研究方面崭露头角
- **其他高校**：各地高校的图形学和 CAD 方向研究者形成了更广泛的参与群体

## 14.5 研究格局的变化

从八十年代的"三角"（复旦—浙大—山大），到2000年代的"多中心"，中国计算几何的研究格局完成了从集中到分散的演变。这是一种健康的生态，意味着学科的根系更深、更广。
