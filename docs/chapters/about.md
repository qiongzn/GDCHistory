# 关于本书的写作方式

## 资料来源

本书的内容来自三类来源：

1. **口述访谈**：对亲历者的采访记录，包括当面访谈和书面回忆。凡引用口述内容，均以引用块标注，并注明受访者姓名与访谈时间。
2. **文献资料**：已发表的论文、教材、会议论文集、以及各机构保存的内部资料。
3. **编者整理**：在口述和文献基础上的叙述性整理，力求忠实但不追求逐字核实。

## 图片说明

本书收录了大量历史照片、手绘草图、系统截图和几何图形。图片来源说明见附录《图片与资料来源说明》。

图片均支持**点击放大**查看细节。

## 关于准确性

本书是回忆录，不是论文。时间、地点、细节可能存在不同亲历者的不同记忆版本。遇到有争议的事实，编者会在注释中标注，欢迎知情者提供更准确的版本。
