# 附录一　专委会沿革

## 全国计算几何协作组（1982–2001）

| 时间 | 事件 |
|------|------|
| 1982 | 青岛短训班，协作组雏形形成 |
| 1983 | 第一届全国计算几何学术会议 |
| 1986 | 协作组正式确立，复旦—浙大—山大三方格局稳定 |
| 1990s | 低谷期，年会坚持举办，参与规模有所收缩 |
| 1998–2000 | 新一代学者逐步接班，讨论重组 |

## 几何设计与计算专委会 GDC（2001–至今）

| 时间 | 事件 |
|------|------|
| 2001 | GDC 专委会正式成立，挂靠 CSIAM |
| 2002 | 第一届全国几何设计与计算学术会议（GDC 会议）|
| 2006 | GDC 会议规模扩大，参与单位超过 20 所 |
| 2014 | SIGGRAPH Asia 2014 深圳，GDC 社区积极参与 |
| 2020s | 工业软件议题进入 GDC 核心议程 |
