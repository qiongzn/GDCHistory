# 附录五　图片与资料来源说明

## 图片使用说明

本书收录的图片来自以下几类来源：

1. **历史照片**：由相关机构和个人提供，已获授权使用。具体来源见各图图注。
2. **几何图形**：由编委会或章节作者绘制，使用开源工具（如 GeoGebra、Matplotlib）生成。
3. **占位图**：部分章节图片位置目前使用占位符，待后续补充实际图片。
4. **引用图片**：来自已发表文献的图片，已注明原始出处，如需转载请联系原作者。
