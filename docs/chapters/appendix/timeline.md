# 附录二　重要会议与时间轴

## 大事年表

<ul class="timeline">
  <li><strong>1960s–1970s</strong> — 造船、航空工业的几何需求推动数学家"下厂"，计算几何问题在中国工业现场萌芽</li>
  <li><strong>1978</strong> — 改革开放，高校科研秩序恢复，海外学术交流重启</li>
  <li><strong>1981</strong> — 梁友栋、汪嘉业等海归学者回国，带回国际计算几何最新进展</li>
  <li><strong>1982</strong> — 全国计算几何短训班，青岛举办；全国计算几何协作组酝酿成立</li>
  <li><strong>1983</strong> — 第一届全国计算几何学术会议</li>
  <li><strong>1986</strong> — 协作组框架稳定，浙大—山大—复旦三角格局形成</li>
  <li><strong>1990s初</strong> — 出国潮冲击，人才断层，学科进入低谷期</li>
  <li><strong>1995前后</strong> — Chinagraph 起步，计算机图形学兴起，计算几何寻求新方向</li>
  <li><strong>2001</strong> — 几何设计与计算专委会（GDC）正式成立</li>
  <li><strong>2002</strong> — 第一届全国几何设计与计算会议（GDC 年会）</li>
  <li><strong>2006前后</strong> — 新一代学者（清华、中科大等）全面进入，研究格局多中心化</li>
  <li><strong>2014</strong> — SIGGRAPH Asia 2014 深圳举办，中国图形学国际地位显著提升</li>
  <li><strong>2015后</strong> — "卡脖子"议题兴起，工业软件受到国家战略重视，计算几何的工程价值重新凸显</li>
  <li><strong>2020s</strong> — AI 与几何计算融合，神经网络几何表示兴起，领域进入新一轮变革</li>
  <li><strong>2025</strong> — 本书写作时间，记录四十年历程</li>
</ul>
