# 第19章　四十年，我们留下些什么

> "一代人有一代人的使命。我们这代人的使命，是把这门学科在中国种下来。"

---

## 19.1 方法、系统与人

四十年，中国计算几何留下了三类遗产：

**方法**：从样条插值到 B 样条造型，从 Bézier 曲面到细分方法，从参数化到网格处理——一套完整的几何计算工具箱，在中国被引进、消化、发展和创新。

**系统**：从最早的数控编程辅助工具，到1980年代的原型 CAD 系统，到2000年代的商业软件——中国有了自己的几何计算系统，尽管与国际最高水平仍有差距。

**人**：这是最重要的遗产。几十年间，中国培养了数以千计的计算几何研究者和工程师，他们分布在高校、研究院所和企业，构成了这个领域在中国存续和发展的根本力量。

## 19.2 几代学者的接力

这四十年，大致可以分为三代：

<ul class="timeline">
  <li><strong>第一代（1980年代）</strong>——奠基者。苏步青、刘鼎元、梁友栋、汪嘉业……他们在几乎一无所有的条件下建立了学科的基本框架。</li>
  <li><strong>第二代（1990–2000年代）</strong>——守护者与扩展者。在低谷中坚守，在新技术浪潮中调适，完成了从协作组到 GDC 专委会的制度重建。</li>
  <li><strong>第三代（2010年代至今）</strong>——国际化一代。在更广阔的国际舞台上工作，将中国计算几何带入全球学术主流，同时开始直面工业软件的产业挑战。</li>
</ul>

## 19.3 薪火相传的真义

"薪火相传"这个词，用在这里不是修辞，是字面的真实：一代人点燃的火，被下一代人接过来，继续燃烧，传给更下一代。

这中间有断点，有困难，有迷失，有失望。但火没有灭。
