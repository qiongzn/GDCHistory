# 第20章　写给后来者

> "中国计算几何的下一个四十年，将由你们来写。"

---

## 20.1 工业需求仍在

如果你是一位刚刚进入计算几何或相关领域的年轻学者，有一件事值得你知道：**这个领域从未脱离过真实的工业需求**。

1960年代是船体放样和飞机蒙皮；1980年代是数控编程和 CAD 系统；2000年代是数字制造和逆向工程；2020年代是工业软件内核、增材制造、数字孪生……需求在变，但"用数学方法精确处理几何问题"这一核心任务从未消失。

你不需要走进工厂，但你需要知道工厂在哪里，它需要什么。

## 20.2 基础理论仍然重要

同样值得你知道的是：**基础理论从未过时**。

每一波应用的浪潮，到最后都会遇到理论的瓶颈。样条理论、逼近论、代数几何、微分几何——这些看似"无用"的数学工具，往往在意想不到的时刻成为解决工程难题的关键。

打好数学基础，不是迂腐，是投资。

## 20.3 中国计算几何的下一个四十年

2025年，中国计算几何站在一个新的起点上：
- 学术实力已经与国际同步乃至局部领先
- 工业软件的国产化面临历史性机遇
- 人工智能正在与几何计算深度融合，带来全新的问题和方法

下一个四十年，会比前四十年更精彩，也更艰难。

前人种树，后人乘凉。但乘凉的人，也要种自己的树。
